My custom cards for [`EDOPro`](https://projectignis.github.io/index.html).

# Download

Add this entry to the `repos` field of `<edopro-dir>/config/user_configs.json`, where `edopro-dir` is the directory containing the executable `EDOPro`:

    {
        "url": "https://github.com/fopancaldi/edopro_custom_cards",
        "repo_name": "fopancaldi's custom cards",
        "repo_path": "./repositories/fopancaldi",
        "should_update": true,
        "should_read": true
    }

# Art

The card pictures were made with [`ygocarder`](https://lauqerm.github.io/ygocarder/).
